## oh-my-cloud-skills

A Claude Code plugin marketplace for AWS cloud work.

### Plugins

| Plugin | Purpose |
|--------|---------|
| aws-content-plugin | Content creation (presentations, diagrams, docs, workshops) |
| aws-ops-plugin | Infrastructure operations & troubleshooting |
| kiro-power-converter | Convert Claude Code plugins to Kiro Power format |

### Quick Install

```bash
# Install via Claude Code plugin marketplace
/plugin marketplace add aws-content-plugin
/plugin marketplace add aws-ops-plugin
```
