```markdown
## Traditional Ops vs AIOps

### Traditional Ops
- 수동 모니터링 & 대시보드 확인
- 정적 임계값 기반 알림
- 사후 대응 (Reactive)
- MTTR: **4시간 이상**

### AIOps
- 자동화된 이상 탐지
- ML 기반 동적 기준선
- 사전 예방 (Proactive)
- MTTR: **30분 이하**
```

## 주요 기능

| 기능 | 설명 |
|----|------|
| `{.click}` Fragment | 스페이스바로 요소를 순차적으로 표시 |
| 다크 테마 | AWS Squid Ink (#232F3E) 배경 |
| 16:9 비율 | FHD/4K 해상도 자동 스케일링 |
| 진행 표시줄 | 하단에 현재 슬라이드 위치 표시 |
| 프레젠터 뷰 | P 키로 노트/타이밍 확인 |
| 개요 모드 | O 키로 전체 슬라이드 그리드 보기 |

## 키보드 단축키

| 키 | 동작 |
|----|------|
| ← → | 이전/다음 슬라이드 |
| Space | 다음 Fragment 또는 다음 슬라이드 |
| F | 전체 화면 |
| P | 프레젠터 뷰 |
| O | 개요 모드 |
| Esc | 종료 |
